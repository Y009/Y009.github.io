§ timer_checkFlag()

void timer_checkFlag ( )

Checks whether there has been a timer interrupt and increments a counter if so.

Is called from application level in every cycle. timerFlag is set to true whenever a timer A interrupt has occurred. Is used to get a delay of TIMETO * interruptCounter.
