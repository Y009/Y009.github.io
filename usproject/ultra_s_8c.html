§ ultraS_ISR()

__interrupt void ultraS_ISR ( void  )

Interrupt vector for Port 1.

Is triggered when the hardware ultrasonic module sends back an echo signal.
The rising edge of the echo signal is saved as the starting time and the falling edge as the ending time.

§ ultraS_prepInfo()

void ultraS_prepInfo ( )

Calculates measured distance.

Measured distance is calculated as: end time - start time, which is then divided by 58 - constant to get the distance in centimeters.
End time is the difference of the end and start time multiplier times the counter size added to the measured end time. Start time is just the measured start time.

Attention
Only when rollover happens!
Adds the entire counter size to endTime to compensate for potential rollover.
