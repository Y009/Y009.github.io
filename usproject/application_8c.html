§ application_cyclic()

void application_cyclic ( )

The main cyclic function to control the workflow.

Initiates check on button.
If the button has been debounced starts the work (if not already in progress.)
Clears the display and checks with ultrasonic module.
Sends a string to the LCD module to be displayed based on value from the ultrasonic module.

§ convertUDec()

void convertUDec ( unsigned long  n)

Converts integer value to string

Converts integer to string and places it in String to the appropriate position
Converts each digit of the value on its own by taking modulo 10 of it and adding a value of 30 to it.
Based on the ascii table where numbers are from 30 to 39.

Attention
Currently limited to 4 digit numbers.

nNumeric value to be converted to string.
